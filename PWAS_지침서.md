## 1. 프로그램 개요

### 목적
- PSP/PPP 패널식 기대기옹벽의 **기존 구조물 안전성·안정성 평가** 전용 도구
- **설계 프로그램이 아님** — 현장에 이미 설치된 구조물을 진단·평가
- 기존 구조계산서가 없는 현장에서 KDS 기준으로 역산 평가 가능
- 출력 결과는 진단기술사의 최종 확인이 필요함 (면책 명시)

## 3. 공법 이해

### 3-1. 현장 적용 빈도
- **혼용 (PSP+PPP) ★★★★★** — 현장 최다
  - 상부 단: 소일네일 (토압 작음, 경제적)
  - 하부 단: 영구앵커 (토압·수압 집중, 안정 지반까지 정착 필요)
  - 중간 단: 지반 변화에 따라 전환
- **PSP 단독 ★★★** — H ≤ 6m, 지반 양호한 경우
- **PPP 단독 ★★** — 지반 불량, 배면에 중요 구조물 있는 경우

### 3-2. 소일네일 vs 영구앵커 상세 비교
| 항목 | 소일네일 (PSP) | 영구앵커 (PPP) |
|------|--------------|--------------|
| 재료 | 강관 (hollow bar) | 강관 또는 강연선 |
| 길이 | ~0.65 × H | ~1.4 × H |
| 경사 | 수평 아래 10~15° (도면: 12°) | 수평 아래 15~25° (도면: 20°) |
| 단면 구성 | 전 구간 그라우팅 — 단일 바디 | 자유장(unbonded) + 정착장(bonded) |
| 두부 | 소형 지압판 + 너트 | 대형 지압판 + 쐐기/너트 블록 |
| 삽도 표현 | 주황 실선 (그라우팅 바디) | 파선(자유장) + 굵은 실선+캡(정착장) |
| 인장 방식 | 수동 (그라우트 부착력) | 능동 (프리스트레스 도입) |
| 적용 기준 | KDS 11 70 15 소일네일링 | KDS 11 70 15 앵커 |

### 3-3. 시공 방법
| 방법 | 설명 | 특징 |
|------|------|------|
| Top-down | 상부 → 하부 순서로 굴착·시공 | PSP/PPP 일반적 방법 |
| Bottom-up | 하부 기초 먼저 → 상향 시공 | 일부 현장 적용 |

## 4. 구조물 형태 및 시공 순서

### 4-1. 시공 순서 (업체 시공도 확인 사항)
각 단마다 다음을 반복:
1. **레벨링 콘크리트 타설** — 해당 단의 독립 기초
2. **기초 상부에 패널 설치**
3. **뒤채움 (골재 또는 콘크리트)**
4. **전면 인장작업 (네일 또는 앵커 긴장)**

→ 이 과정이 최하단부터 위로 반복 (Top-down의 경우 최상단부터)

### 4-2. 단면 구성 (확정)
```
지표면 (비탈면)
  │
  ├─ [1단] 패널 n₁장 (수직 직사각형)
  │        ← 레벨링 콘크리트 (1단 독립 기초)
  │        보강재: 패널 1장당 네일 또는 앵커 1개
  │  소단(berm) 폭 b₁
  │
  ├─ [2단] 패널 n₂장
  │        ← 레벨링 콘크리트 (2단 독립 기초)
  │        보강재: 패널 1장당 네일 또는 앵커 1개
  │  소단 폭 b₂
  │
  ├─ [3단] ...
  │
  └─ [최하단] 패널 n_last장
             ← 레벨링 콘크리트 (최하단·가장 두꺼움)
```

### 4-3. 핵심 확정 사항
| 항목 | 내용 | 근거 |
|------|------|------|
| 레벨링 콘크리트 | **각 단마다 독립 기초** (최하단만이 아님) | 시공순서도 확인 |
| 보강재 배치 | **패널 1장당 1개** (단당 n장이면 n개) | 시공도 확인 |
| 패널 형태 | **수직 직사각형** (경사 없음) | 실제 사진 확인 |
| 소단 역할 | 시공 작업공간 + 구조적 단차 | — |
| 기초 형식 | 단별 독립 → **단별 독립 해석** | 시공순서 확인 |

### 4-4. 비탈면 경사선의 실제 형태
직선이 아니라 **소단마다 꺾이는 계단형**:
```
1단 배면 상단
  → (수평, b₁만큼) → 소단 끝
  → (θ 경사) → 2단 배면 상단
  → (수평, b₂만큼) → 소단 끝
  → (θ 경사) → ...
  → 지반선
```
이 형상이 Coulomb 쐐기의 배면 경사 β에 해당 → 토압 계산에 직접 사용됨

## 5. 해석 개념 (확정)

### 5-1. 해석 단위
| 검토 항목 | 해석 단위 | 근거 |
|-----------|-----------|------|
| **외적안정** (활동·전도·지지력) | **단별 독립 검토** | 각 단 자체 레벨링 기초 위에서 독립 저항 |
| **보강재 → 패널 전달** | 단별 독립 | 수평 토압을 각 단 보강재가 독립 저항 |
| **전체안정 (원호활동)** | 전 구간 통합 | SLOPE/W 등 외부 프로그램 사용 |

### 5-2. 각 입력값의 해석 영향

#### 단별 높이 h_i — 가장 중요한 변수
```
Coulomb 주동토압:
  Pa_i = ½ × γ × h_i² × Ka

h_i = 단 i의 패널 수 × 패널 1장 높이

→ h_i가 다르면 Pa_i가 달라짐
→ 각 단 보강재(네일/앵커)의 요구 인장력이 달라짐
→ 활동·전도·지지력 FS가 달라짐
```

#### 소단 폭 b_i — 토압 및 안정 검토에 반영
```
① 배면토 자중 (전도 저항에 기여):
   W_berm = γ × b_i × h_i
   → 전도 저항 모멘트 증가 (유리)
   → 지지력 검토 수직력 증가

② 배면 형상 (Coulomb 쐐기):
   소단이 있으면 배면이 계단형 → 시행쐐기법 적용 시 형상 반영

③ 상부 단 → 하부 단 하중 전달 가능성:
   FHWA NHI-14-007 기준:
   소단 폭 D < 2 × h_하단 이면 상재하중 전달 가능
   PSP/PPP 소단 폭 (0.3~0.6m)은 대부분 2H 미만
   → Phase 03/04 설계 시 상재하중 q_i 반영 여부 결정 필요

④ 전체안정 (원호활동):
   소단 형상이 파괴면 형상 결정에 영향
```

#### 비탈면 경사각 θ — 토압 계수 Ka에 직접 반영
```
Coulomb 토압 계수:
  Ka = f(φ, θ, δ, β)
         ↑
         배면 경사각 β = θ (비탈면 각도)

θ가 달라지면 → Ka 달라짐 → Pa 달라짐
```

#### 기초 폭 B (Phase 03 입력, 핵심)
```
활동 검토:
  마찰 저항력 = (ΣV) × tanδ
  → 기초 저면 마찰면 = B × L

전도 검토:
  합력 편심 e = B/2 - (저항모멘트 - 전도모멘트) / ΣV
  → e < B/6 조건

지지력 검토:
  유효 기초 폭 B' = B - 2e
  → B가 클수록 지지력 유리
```

### 5-4. PC 패널 구조 해석 원칙 (2026-05-09 확정)

> Claude Opus 구조공학 자문 + KDS/FHWA 기준 검토 결과. **변경 금지.**

#### ① 패널 구조 분류 — 부분 프리스트레스트 콘크리트 (PPC)

PSP/PPP 패널은 **RC가 아닌 PC(프리스트레스트 콘크리트) 구조**로 검토해야 한다.

- 보강재 두부(head)에서 도입되는 긴장력 T_res = **포스트텐션 외부 정착 형식의 능동 프리스트레스**
- T_res(= P_eff)가 패널 단면에 축압축으로 작용 → 균열모멘트 M_cr 증가, 사용성 개선
- 단순 RC 검토 시: M_cr 과대평가, 유효 A_ps 누락 → **부적절**
- **적용 기준:** KDS 14 30(PC 우선) + KDS 14 20(일반 RC 병용)

| 검토 항목 | 적용 식 | KDS 조항 |
|---|---|---|
| 균열모멘트 | M_cr = (f_r + P_eff/A_c + P_eff·e·c/I_c)·I_c/c | KDS 14 30 4.3.2 |
| 사용성 허용응력 | σ_t ≤ 0.5√f_ck | KDS 14 30 4.3.1 |
| 극한휨강도 | M_n = A_ps·f_ps·(d_p−a/2) + A_s·f_y·(d−a/2) | KDS 14 30 4.2.1 |
| 유효 PS력 | P_eff = P_i − ΔP (즉시손실 + 시간의존손실) | KDS 14 30 4.4 |

> **Lift-off Test 실측 T_res = P_eff 직접 입력.** Phase 02 → Phase 03 이월 흐름이 이 원칙과 정합함.

#### ② 해석 단위 — 패널 1장 (B_panel × h_panel)

**단위 m(m당 해석폭 b=1000mm) 해석은 이 공법에 부적절하다.**

| 구분 | 단위 m 해석 (부적절) | 패널 1장 해석 (올바름) |
|---|---|---|
| 연속성 가정 | 횡방향 연속 슬래브 | 독립 패널 — 연속성 없음 |
| 하중 단위 | m당 | B_panel당 |
| 기준 | KDS 14 20 §6.5 (RC 슬래브) | FHWA NHI-14-007 §5.3 + KDS 14 30 |

- **보강재 1개 = 패널 1장** → 하중 분배 단위가 B_panel
- **지지조건:** 4변 자유 + 중앙 집중력(보강재 두부) + 배면 등분포 토압
- **형상비에 따른 해석 분기:**
  - B_panel / h_panel ≤ 2 → 평판(plate) 해석 (Westergaard/Pucher 계수)
  - B_panel / h_panel > 2 → 보(beam) 해석으로 축약
- **FHWA C_F 보정계수:** 시공 편차·하중 분배 불확실성 반영 (1.0~2.0 범위)

따라서 Phase 03 A섹션 `b` 필드는:
- ~~해석 단위폭 1000mm(단위 m)~~
- **B_panel: 패널 1장의 실제 폭 (mm)** — 제조사 사양서 확인

#### ③ 두부 하중 전달 경로 — 3단 검토 필수

```
T_res → 지압판(A_bp) → 패널 전면 집중하중
  ├─ [A] 펀칭전단  V_u ≤ φ·(1/3)√f_ck·b_0·d       [KDS 14 20 4.2.10]
  ├─ [B] 패널 휨   φM_n ≥ M_u                       [KDS 14 30 4.2.1]
  └─ [C] 정착지압  f_b = T_res/A_bp ≤ 0.7f_ci·√(A/A_bp) [KDS 14 30 4.5.2]
```

- **[A] 펀칭전단** — 지압판 주변 임계둘레 b_0 기준, PS 압축응력 기여 포함
- **[B] 패널 휨** — 4변 자유·중앙 집중지지 평판, 배면 등분포 토압 동시 작용
- **[C] 정착지압** — 지압판 면적, f_ci, 분할인장(splitting) 보강근 확인

#### ④ Phase 04-B/C 개발 시 계산 모듈 구조 (예정)

```
src/calc/
  panel/
    geometry.ts     B_panel, h_panel, t, 편심 e
    psForce.ts      P_eff = T_res (Lift-off) 또는 P_i × (1−손실율)
    flexure.ts      M_cr, M_n (KDS 14 30)
    punching.ts     V_c (KDS 14 20 + FHWA)
    bearing.ts      f_b (KDS 14 30 4.5.2)
    plate.ts        4변 자유·중앙지지 평판 모멘트계수 표
```

### 5-3. 외적안정 검토 흐름 (단별)
```
단 i에 대해:
  1) h_i 확인 (패널 수 × 패널 높이)
  2) 배면토 단위중량 γ, 내부마찰각 φ 입력
  3) Ka 계산 (Coulomb)
  4) Pa_i = ½γh_i²Ka (+ 상재하중 항)
  5) 활동 FS_i = (ΣV×tanδ + c×B) / Pa_i×cosα
  6) 전도 FS_i = ΣM_저항 / ΣM_전도
  7) 지지력 FS_i = q_u / q_max
  8) FS 기준 대비 → 합격(✓) / 불합격(✗)
```

### 6-1. 입력 방식 — B방식 채택 (자유 입력)

- 단별 패널 수 + 소단 폭을 자유 입력

### Phase 04 — 안정성 검토 🔲 핵심 미구현

#### A. 외적안정 [KDS 11 80 20] — 단별 독립 검토
```
각 단 i에 대해 (정상하중, 내진검토 제외):
  토압: Pa_i = ½γh_i²Ka (Coulomb)
  활동: FS = (ΣV×tanδ + c×B) / ΣH  ≥ 1.5
  전도: FS = ΣM_저항 / ΣM_전도      ≥ 1.5
  지지력: FS = q_u / q_max           ≥ 2.5
  자체파괴: FS ≥ 2.0
```

#### B. 보강재 → 패널 전달 [KDS 11 70 15 + FHWA NHI-14-007 §5.3]

**해석 단위: 패널 1장 (B_panel × h_panel)** — 단위 m 해석 부적합 (§5-4 참조)

```
  ① T_res 확정 (Lift-off 실측 우선, 추정 시 T_0 × (1−손실율))

  ② 펀칭전단 [KDS 14 20 4.2.10]
     V_u = T_res
     V_c = φ · (1/3)√f_ck · b_0 · d
     b_0 = 2(c_1 + c_2 + 2d)  ← 지압판 주변 임계단면 둘레

  ③ 패널 휨 [KDS 14 30 4.2.1] — 4변 자유·중앙 집중지지 평판
     M_u = C_F · T_res · (배면 등분포 토압 항)  ← FHWA C_F 보정
     M_n = A_ps·f_ps·(d_p−a/2) + A_s·f_y·(d−a/2)
     요구: φM_n ≥ M_u   (φ = 0.85)
```

#### C. PC 패널 단면 검토 [KDS 14 30 우선 + KDS 14 20 병용]

**구조 분류: 부분 프리스트레스트 콘크리트 (PPC)** — RC 단독 검토 불가 (§5-4 참조)

```
  ① 유효 PS력
     P_eff = P_i − ΔP (즉시손실: 마찰·탄성단축 / 시간의존: 건조수축·크리프·릴랙세이션)
     또는 P_eff = T_res (Lift-off 실측)  ← 실측 우선

  ② 균열모멘트 [KDS 14 30 4.3.2]
     M_cr = (f_r + P_eff/A_c + P_eff·e·c/I_c) · I_c/c

  ③ 극한휨강도 [KDS 14 30 4.2.1]
     M_n = A_ps·f_ps·(d_p−a/2) + A_s·f_y·(d−a/2)

  ④ 전단강도 [KDS 14 20 6.4]
     V_n = V_c + V_s

  ⑤ 정착부 지압응력 [KDS 14 30 4.5.2]
     f_b = T_res / A_bp ≤ 0.7·f_ci·√(A/A_bp) ≤ 1.4·f_ci
```

#### D. 전체안정 (원호활동) — 외부 입력


  - 치수선: H(전체), 최하단 단 높이
  - 라벨: n=단수(총 패널수), θ=경사각, 공법명, 시공방법 뱃지
- 하단 요약 바: H(역산) / L / n / 총패널 / θ / 공법 / 시공

### 10-4. 모식도 읽는 법
```
좌측 치수선:  H = 전체 높이 (역산)
              각 단 높이 = 패널수 × 패널높이
상단 라벨:    n=4 (총 8장) → 4단, 각 단 평균 2장
경사선:       꺾인 실선 = 비탈면 실제 형상 (θ각도)
파랑 보강재:  PPP 영구앵커 (파선=자유장, 실선=정착장)
주황 보강재:  PSP 소일네일 (주황 실선 = 그라우팅)
회색 기초:    단별 레벨링 콘크리트 (최하단 두꺼움)
```
