## 1. 프로그램 개요

### 목적
- PSP/PPP 패널식 기대기옹벽의 **기존 구조물 안전성·안정성 평가** 전용 도구
- **설계 프로그램이 아님** — 현장에 이미 설치된 구조물을 진단·평가
- 기존 구조계산서가 없는 현장에서 KDS 기준으로 역산 평가 가능
- 출력 결과는 진단기술사의 최종 확인이 필요함 (면책 명시)

- **배포 URL:** https://sjidok750-creator.github.io/rewall/
- **로컬 경로:** D:/workclaude/rewall/
- **기술 스택:** React + TypeScript + Vite
- **배포 방법:** main 브랜치 push → GitHub Actions 자동 빌드 → GitHub Pages

## 3. 공법 이해

### 3-1. 현장 적용 빈도
- **혼용 (PSP+PPP) ★★★★★** — 현장 최다
  - 상부 단: 소일네일 (토압 작음, 경제적)
  - 하부 단: 영구앵커 (토압·수압 집중, 안정 지반까지 정착 필요)
  - 중간 단: 지반 변화에 따라 전환
- **PSP 단독 ★★★** — H ≤ 6m, 지반 양호한 경우
- **PPP 단독 ★★** — 지반 불량, 배면에 중요 구조물 있는 경우

### 3-2. 소일네일 vs 영구앵커 상세 비교
| 항목 | 소일네일 (PSP) | 영구앵커 (PPP) |
|------|--------------|--------------|
| 재료 | 강관 (hollow bar) | 강관 또는 강연선 |
| 길이 | ~0.65 × H | ~1.4 × H |
| 경사 | 수평 아래 10~15° (도면: 12°) | 수평 아래 15~25° (도면: 20°) |
| 단면 구성 | 전 구간 그라우팅 — 단일 바디 | 자유장(unbonded) + 정착장(bonded) |
| 두부 | 소형 지압판 + 너트 | 대형 지압판 + 쐐기/너트 블록 |
| 삽도 표현 | 주황 실선 (그라우팅 바디) | 파선(자유장) + 굵은 실선+캡(정착장) |
| 인장 방식 | 수동 (그라우트 부착력) | 능동 (프리스트레스 도입) |
| 적용 기준 | KDS 11 70 15 소일네일링 | KDS 11 70 15 앵커 |

### 3-3. 시공 방법
| 방법 | 설명 | 특징 |
|------|------|------|
| Top-down | 상부 → 하부 순서로 굴착·시공 | PSP/PPP 일반적 방법 |
| Bottom-up | 하부 기초 먼저 → 상향 시공 | 일부 현장 적용 |

## 4. 구조물 형태 및 시공 순서

### 4-1. 시공 순서 (업체 시공도 확인 사항)
각 단마다 다음을 반복:
1. **레벨링 콘크리트 타설** — 해당 단의 독립 기초
2. **기초 상부에 패널 설치**
3. **뒤채움 (골재 또는 콘크리트)**
4. **전면 인장작업 (네일 또는 앵커 긴장)**

→ 이 과정이 최하단부터 위로 반복 (Top-down의 경우 최상단부터)

### 4-2. 단면 구성 (확정)
```
지표면 (비탈면)
  │
  ├─ [1단] 패널 n₁장 (수직 직사각형)
  │        ← 레벨링 콘크리트 (1단 독립 기초)
  │        보강재: 패널 1장당 네일 또는 앵커 1개
  │  소단(berm) 폭 b₁
  │
  ├─ [2단] 패널 n₂장
  │        ← 레벨링 콘크리트 (2단 독립 기초)
  │        보강재: 패널 1장당 네일 또는 앵커 1개
  │  소단 폭 b₂
  │
  ├─ [3단] ...
  │
  └─ [최하단] 패널 n_last장
             ← 레벨링 콘크리트 (최하단·가장 두꺼움)
```

### 4-3. 핵심 확정 사항
| 항목 | 내용 | 근거 |
|------|------|------|
| 레벨링 콘크리트 | **각 단마다 독립 기초** (최하단만이 아님) | 시공순서도 확인 |
| 보강재 배치 | **패널 1장당 1개** (단당 n장이면 n개) | 시공도 확인 |
| 패널 형태 | **수직 직사각형** (경사 없음) | 실제 사진 확인 |
| 소단 역할 | 시공 작업공간 + 구조적 단차 | — |
| 기초 형식 | 단별 독립 → **단별 독립 해석** | 시공순서 확인 |

### 4-4. 비탈면 경사선의 실제 형태
직선이 아니라 **소단마다 꺾이는 계단형**:
```
1단 배면 상단
  → (수평, b₁만큼) → 소단 끝
  → (θ 경사) → 2단 배면 상단
  → (수평, b₂만큼) → 소단 끝
  → (θ 경사) → ...
  → 지반선
```
이 형상이 Coulomb 쐐기의 배면 경사 β에 해당 → 토압 계산에 직접 사용됨

## 5. 해석 개념 (확정)

### 5-1. 해석 단위
| 검토 항목 | 해석 단위 | 근거 |
|-----------|-----------|------|
| **외적안정** (활동·전도·지지력) | **단별 독립 검토** | 각 단 자체 레벨링 기초 위에서 독립 저항 |
| **보강재 → 패널 전달** | 단별 독립 | 수평 토압을 각 단 보강재가 독립 저항 |
| **전체안정 (원호활동)** | 전 구간 통합 | SLOPE/W 등 외부 프로그램 사용 |

### 5-2. 각 입력값의 해석 영향

#### 단별 높이 h_i — 가장 중요한 변수
```
Coulomb 주동토압:
  Pa_i = ½ × γ × h_i² × Ka

h_i = 단 i의 패널 수 × 패널 1장 높이

→ h_i가 다르면 Pa_i가 달라짐
→ 각 단 보강재(네일/앵커)의 요구 인장력이 달라짐
→ 활동·전도·지지력 FS가 달라짐
```

#### 소단 폭 b_i — 토압 및 안정 검토에 반영
```
① 배면토 자중 (전도 저항에 기여):
   W_berm = γ × b_i × h_i
   → 전도 저항 모멘트 증가 (유리)
   → 지지력 검토 수직력 증가

② 배면 형상 (Coulomb 쐐기):
   소단이 있으면 배면이 계단형 → 시행쐐기법 적용 시 형상 반영

③ 상부 단 → 하부 단 하중 전달 가능성:
   FHWA NHI-14-007 기준:
   소단 폭 D < 2 × h_하단 이면 상재하중 전달 가능
   PSP/PPP 소단 폭 (0.3~0.6m)은 대부분 2H 미만
   → Phase 03/04 설계 시 상재하중 q_i 반영 여부 결정 필요

④ 전체안정 (원호활동):
   소단 형상이 파괴면 형상 결정에 영향
```

#### 비탈면 경사각 θ — 토압 계수 Ka에 직접 반영
```
Coulomb 토압 계수:
  Ka = f(φ, θ, δ, β)
         ↑
         배면 경사각 β = θ (비탈면 각도)

θ가 달라지면 → Ka 달라짐 → Pa 달라짐
```

#### 기초 폭 B (Phase 03 입력, 핵심)
```
활동 검토:
  마찰 저항력 = (ΣV) × tanδ
  → 기초 저면 마찰면 = B × L

전도 검토:
  합력 편심 e = B/2 - (저항모멘트 - 전도모멘트) / ΣV
  → e < B/6 조건

지지력 검토:
  유효 기초 폭 B' = B - 2e
  → B가 클수록 지지력 유리
```

### 5-3. 외적안정 검토 흐름 (단별)
```
단 i에 대해:
  1) h_i 확인 (패널 수 × 패널 높이)
  2) 배면토 단위중량 γ, 내부마찰각 φ 입력
  3) Ka 계산 (Coulomb, 지진 시 M-O)
  4) Pa_i = ½γh_i²Ka (+ 상재하중 항)
  5) 활동 FS_i = (ΣV×tanδ + c×B) / Pa_i×cosα
  6) 전도 FS_i = ΣM_저항 / ΣM_전도
  7) 지지력 FS_i = q_u / q_max
  8) FS 기준 대비 → 합격(✓) / 불합격(✗)
```

## 6. 입력값 구조

### 6-1. 입력 방식 — B방식 채택 (자유 입력)
패널 구성으로 H를 결정하는 방식:
- 단별 패널 수 + 소단 폭을 자유 입력
- **H = 역산** (총 패널 수 × 패널 1장 높이)
- 현장에서 패널 수를 먼저 파악하고 H를 역산하는 실무에 부합

**A방식(H 고정)을 채택하지 않은 이유:**
기존 구조물 평가 현장에서는 H보다 패널 수를 먼저 확인하는 경우가 많음

### 6-2. 타입 정의 (WallSchematic.tsx)
```typescript
// 단 1개의 설정
export interface TierConfig {
  panels: number      // 이 단의 패널 수 (1~10)
  bermWidth: number   // 이 단 하부 소단 폭 (m) — 최하단은 미사용
}

// 전체 옹벽 파라미터
export interface WallParams {
  height: number        // 역산된 전체 H (m) = Σ(panels_i) × panelHeight

  slopeAngle: number    // 비탈면 경사각 θ (°)
  wallThick: number     // 패널 두께 t (m)
  method: 'PSP' | 'PPP' | 'mixed' | ''
  construction: 'top-down' | 'bottom-up' | 'unknown' | ''
  panelHeight: number   // 패널 1장 높이 (m, 전 단 공통)
```

| 단 번호 | — | 자동 | 1=최상단 |
| 패널 수 | 장 | 1~10 | 직접 입력 |
| 소단 폭 | m | 0.1~3.0 | 최하단은 '—' |
| 단 높이 | m | 자동 | 패널수 × 패널높이 |
| **합계 H** | m | — | 역산 표시 |

**단수 변경 시 동작:**
- 늘어날 때: 기존 값 유지 + 새 단 기본값(2장, 0.5m) 추가
- 줄어들 때: 하단 단부터 제거 (기존 값 유지 불가)

## 7. 평가 흐름 6 Phase

### Phase 01 — 기본정보 수집 ✅ 구현완료
공법분류 / 자료보유현황 / KDS버전 / 시공방법 / 단별 패널·소단 테이블

**출력:** 역산 H, 총 패널 수, 단면 모식도 (실시간 업데이트)

### Phase 02 — 현장조사 🔲 플레이스홀더
- 외관조사: 패널 균열·박리·변위, 두부 부식, 배수공, 기초 세굴
- 비파괴시험: 슈미트해머, 초음파속도, 철근탐사, 탄산화
- 코어 채취: 압축강도 실측
- **Lift-off Test: 앵커·네일 잔존 인장력 실측** ← Phase 03 분기점
- 지반 자료: 시공 당시 보고서 활용
- **★ 단별 레벨링 콘크리트 상태 확인** (균열·세굴·부등침하)

### Phase 03 — 입력값 확정 🔲 플레이스홀더
- 단면·배근 (도면 有/無에 따라 입력 방식 분기)
- 재료강도: 콘크리트 fck, 철근 fy (열화감소율 반영)
- PS 잔존력: Lift-off 실측값 또는 초기긴장력 × 손실율
- **지반정수: γ, φ, c, Ks** ← 토압 계산 핵심
- **앵커·네일 제원: 길이, 직경, 간격, 경사각, 그라우트 강도**
- **기초 폭 B** ← 활동·전도·지지력 계산 핵심
- 하중조건: 상재하중, 지하수위, 지진계수

### Phase 04 — 안정성 검토 🔲 핵심 미구현

#### A. 외적안정 [KDS 11 80 20] — 단별 독립 검토
```
각 단 i에 대해:
  토압: Pa_i = ½γh_i²Ka (Coulomb)
        지진 시: Mononobe-Okabe 적용
  활동: FS = (ΣV×tanδ + c×B) / ΣH  ≥ 1.5 (지진 1.1)
  전도: FS = ΣM_저항 / ΣM_전도      ≥ 1.5 (지진 1.1)
  지지력: FS = q_u / q_max           ≥ 2.5 (지진 2.0)
  자체파괴: FS ≥ 2.0
```

#### B. 보강재 → 패널 전달 [KDS 11 70 15 + FHWA NHI-14-007]
```
  잔존 인장력 T_res 확인
  패널 펀칭전단 검토: V_u / (b_o × d) ≤ φv_c
  패널 휨 검토: 네일 사이 4점지지 단순보 가정
               M_u ≤ φM_n
```

#### C. PC 패널 단면 [KDS 14 20 + KDS 14 30]
```
  PS 유효 잔존력 P_eff 적용
  균열모멘트 M_cr, 극한모멘트 M_n 검토
  전단강도 V_n 검토
  정착부 지압응력 검토
```

#### D. 전체안정 (원호활동) — 외부 입력
```
  SLOPE/W, GeoStudio 등 결과를 수동 입력
  미입력 시 → 경고 플래그, 등급 판정 제한
```

### Phase 05 — 등급 판정 🔲 미구현
- 항목별 FS → 상태 점수
- 외관조사 등급 결합
- 종합 안전등급: **a(정상) / b(모니터링) / c(보수) / d(긴급보수) / e(사용금지)**
- 보수보강 의견 자동 매핑
- 진단기술사 최종 확인 필요 명시

### Phase 06 — 출력 🔲 미구현
- 화면: 항목별 FS 표 + 합격/불합격 + 등급 요약
- PDF 계산서: 전체 입력값, KDS 조항 인용, 계산 과정 상세
- 면책 문구 필수 포함

2. ② 자료 보유현황: 보유 자료 / KDS 버전 선택

4. ④ 단별 제원 테이블:
   - 행: 단 수만큼 자동 생성
   - 열: 단번호 / 패널수(입력) / 소단폭(입력) / 단높이(자동)
   - 하단: 합계 H (역산), 총 패널 수

**우측 모식도 (60%):**
- 입력값 변경 즉시 반영되는 SVG 단면 모식도
- 표시 요소:
  - 패널 (단별 수직 직사각형, 단당 패널수×높이)
  - 레벨링 콘크리트 (단별 독립 기초, 최하단 강조)
  - 소단 (단 경계 수평선, 소단폭 치수 표시)
  - 보강재 (패널 1장당 1개: 네일=주황, 앵커=파랑)
  - 비탈면 경사선 (소단마다 꺾이는 실제 형태)
  - 배면토 해칭 (45° 해칭)
  - 치수선: H(전체), 최하단 단 높이
  - 라벨: n=단수(총 패널수), θ=경사각, 공법명, 시공방법 뱃지
- 하단 요약 바: H(역산) / L / n / 총패널 / θ / 공법 / 시공

### 10-4. 모식도 읽는 법
```
좌측 치수선:  H = 전체 높이 (역산)
              각 단 높이 = 패널수 × 패널높이
상단 라벨:    n=4 (총 8장) → 4단, 각 단 평균 2장
경사선:       꺾인 실선 = 비탈면 실제 형상 (θ각도)
파랑 보강재:  PPP 영구앵커 (파선=자유장, 실선=정착장)
주황 보강재:  PSP 소일네일 (주황 실선 = 그라우팅)
회색 기초:    단별 레벨링 콘크리트 (최하단 두꺼움)
```
