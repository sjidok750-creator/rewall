## 1. 프로그램 개요

### 목적
- PSP/PPP 패널식 기대기옹벽의 **기존 구조물 안전성·안정성 평가** 전용
- 설계 프로그램이 아닌 **현장 진단·평가 도구**
- 기존 구조계산서가 없는 현장에서 KDS 기준으로 역산 평가

### 대상 공법
| 구분 | 전체 명칭 | 구성 |
|------|----------|------|
| PSP | Prestressed Soil-nailed Panel | 소일네일(강관네일) + PS패널 |
| PPP | Prestressed Panel with Permanent anchor | 영구앵커(강관앵커) + PS패널 |
| 혼용 | PSP+PPP 구간별 병행 | 상부 네일 + 하부 앵커 (현장 최다) |

## 3. 구조 이해 — 핵심 개념 (세션 중 확정된 사항)

### 3-1. 공법 선택 빈도 (현장 기준)
- **혼용 (PSP+PPP) ★★★★★** — 현장 최다. 상부 단 네일, 하부 단 앵커
- **PSP 단독 ★★★** — H ≤ 6m, 지반 양호
- **PPP 단독 ★★** — 지반 불량, 배면 중요 구조물

### 3-2. 구조물 형태 (시공순서도 확인)
시공 순서: **레벨링 콘크리트 타설 → 패널 설치 → 뒤채움 → 전면 인장작업** 반복

```
지표면
  ├── [앵커/네일] ─── 1단: 패널 n장 + 레벨링 콘크리트
  ├── [앵커/네일] ─── 2단: 패널 n장 + 레벨링 콘크리트
  ├── [앵커/네일] ─── 3단: 패널 n장 + 레벨링 콘크리트
  └── [앵커/네일] ─── 4단: 패널 n장 + 레벨링 콘크리트(최하단·두꺼움)
```

**핵심 확정 사항:**
- 레벨링 콘크리트 = 각 단마다 독립 기초 (최하단만이 아님)
- 패널 1장당 보강재 1개 (단당 n개가 아님)
- 패널은 수직 직사각형 (경사 없음)
- 각 단 사이 소단(berm) = 시공 작업공간 + 구조적 단차

### 3-3. 소일네일 vs 영구앵커 차이
| 항목 | 소일네일 (PSP) | 영구앵커 (PPP) |
|------|--------------|--------------|
| 길이 | ~0.65×H | ~1.4×H |
| 경사 | 수평 아래 12° | 수평 아래 20° |
| 단면 | 전 구간 그라우팅 (단색 실선) | 자유장(파선) + 정착장(굵은 실선+캡) |
| 두부 | 소형 지압판 + 너트 | 대형 지압판 + 쐐기 블록 |

### 3-4. 해석 단위 (확정)
| 검토 항목 | 해석 단위 | 근거 |
|-----------|-----------|------|
| **외적안정** (활동·전도·지지력) | **단별 독립 검토** | 각 단 자체 레벨링 기초 위에서 독립 저항 |
| **보강재→패널 전달** | 단별 독립 | 수평 토압을 각 단 보강재가 독립 저항 |
| **전체안정 (원호활동)** | 전 구간 통합 | SLOPE/W 등 외부 프로그램 |

### 3-5. 각 입력값의 해석 영향 (중요)

**단별 높이 h_i** — 가장 중요
```
Coulomb 토압: Pa_i = ½ × γ × h_i² × Ka
h_i가 다르면 → Pa_i 달라짐 → 보강재 인장력 달라짐
```

**소단 폭 b_s** — 직접 영향
```
배면토 자중: W_s = γ × b_s × h_i
→ 전도 저항 모멘트 기여 (유리)
→ 지지력 수직력 기여 (유리)
소단 폭이 좁으면 상부 하중 누적 가능성 (FHWA 2H rule)
```

**비탈면 경사각 θ** — 토압 계수에 직접 반영
```
Ka = f(φ, θ, δ, β)  — β = 배면 경사각 (= θ)
→ θ가 달라지면 Ka 달라짐 → Pa 달라짐
```

**기초 폭 B** (Phase 03 입력)
```
활동: 마찰 저항 = N × tanδ × B
전도: 합력 작용점 편심 e = f(B)
지지력: 유효폭 B' = B - 2e
```

### 3-6. 소단 폭과 비탈면 경사선의 관계
비탈면 경사선은 단순 직선이 아니라 **소단마다 꺾이는 계단형**:
```
1단 배면 상단
  → (수평, 소단 폭만큼) → 소단 끝
  → (θ 경사) → 2단 배면 상단
  → (수평) → 소단 끝
  → (θ 경사) → ...
  → 지반선
```
이 형상이 Coulomb 쐐기의 배면 형상 β에 해당 → 토압 계산에 직접 사용

## 4. 입력값 구조 (Phase 01 확정)

### B방식 — 자유 입력 (채택)
- 단별 패널 수 + 소단 폭 자유 입력
- **H = 역산** (총 패널 수 × 패널 높이)
- H를 먼저 입력하는 것이 아니라 패널 구성으로 결정

### TierConfig (단별 설정)
```typescript
interface TierConfig {
  panels: number      // 이 단의 패널 수
  bermWidth: number   // 이 단 하부 소단 폭 (m) — 최하단 미사용
}
```

### WallParams
```typescript
interface WallParams {
  height: number        // 역산된 전체 H (m)
  length: number        // 연장 L (m)
  stages: number        // 단수 n
  slopeAngle: number    // 비탈면 경사각 θ (°)
  wallThick: number     // 패널 두께 t (m)
  method: 'PSP' | 'PPP' | 'mixed' | ''
  construction: 'top-down' | 'bottom-up' | 'unknown' | ''
  panelHeight: number   // 패널 1장 높이 (m, 전 단 공통)
  tiers: TierConfig[]   // 단별 패널 수 + 소단 폭
}
```

## 5. 평가 흐름 (6 Phase)

```
Phase 01 — 기본정보 수집
  공법분류 / 시공방법 / 자료보유현황 / KDS버전
  단별 제원: 패널 수 × 소단 폭 테이블 입력 → H 역산

Phase 02 — 현장조사
  외관조사 / 비파괴시험 / 코어채취 / Lift-off Test / 지반자료
  ★ 단별 레벨링 콘크리트 상태 확인 (균열·세굴·침하)

Phase 03 — 입력값 확정
  단면·배근 / 재료강도 / PS잔존력 / 지반정수
  앵커·네일 제원 (길이·직경·간격·경사각·그라우트 강도)
  하중 조건 (상재하중·지하수위·지진계수)
  기초 폭 B (활동·전도·지지력 계산의 핵심)

Phase 04 — 안정성 검토 ← 핵심 계산
  A. 외적안정 [KDS 11 80 20] — 단별 독립 검토
     Coulomb 토압 (지진: Mononobe-Okabe)
     활동 FS ≥ 1.5 / 전도 FS ≥ 1.5 / 지지력 FS ≥ 2.5 / 자체파괴 FS ≥ 2.0
  B. 보강재→패널 [KDS 11 70 15 + FHWA NHI-14-007]
     잔존인장력 → 펀칭전단 / 패널휨 (4점지지)
  C. PC패널 단면 [KDS 14 20 + 14 30]
     PS잔존력 → 휨강도 / 전단강도 / 정착부
  D. 전체안정 [외부 입력]
     SLOPE/W 결과 수동 입력

Phase 05 — 등급 판정
  항목별 FS + 외관등급 → 종합 a~e등급 → 보수보강 의견

Phase 06 — 출력
  화면 요약 + PDF 계산서 (KDS 조항 추적성 포함)
```

## 11. 신뢰성 원칙

1. 모든 계산식에 KDS 조항번호 명시 (예: KDS 11 80 20 4.3.1 식 (4.3-1))
