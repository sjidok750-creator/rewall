# PWAS — 공법·구조 이해

## 1. 대상 공법

| 구분 | 명칭 | 구성 | 기준 |
|------|------|------|------|
| PSP | Prestressed Soil-nailed Panel | 강관 소일네일 + PS패널 | KDS 11 80 20 + 11 70 15 |
| PPP | Prestressed Panel w/ Permanent anchor | 강관 영구앵커 + PS패널 | 상동 |
| 혼용 | PSP+PPP 구간별 병행 | 상부 네일 + 하부 앵커 | 상동 |

**현장 적용 빈도:** 혼용 ★★★★★ / PSP ★★★ / PPP ★★

---

## 2. 소일네일 vs 영구앵커

| 항목 | 소일네일 (PSP) | 영구앵커 (PPP) |
|------|--------------|--------------|
| 재료 | 강관 hollow bar | 강관 또는 강연선 |
| 길이 | ~0.65 × H | ~1.4 × H |
| 경사 | 수평 아래 12° (10~15°) | 수평 아래 20° (15~25°) |
| 단면 | 전 구간 그라우팅 | 자유장(파선) + 정착장(굵은 실선+캡) |
| 두부 | 소형 지압판 + 너트 | 대형 지압판 + 쐐기 블록 |
| 인장 | 수동 (그라우트 부착력) | 능동 (프리스트레스 도입) |
| 삽도색 | 주황 #D97757 | 파랑 #4A7FA5 |

---

## 3. 시공 순서 (확정 — 업체 시공도 확인)

각 단마다 아래를 반복:
1. **레벨링 콘크리트 타설** → 해당 단의 독립 기초
2. **패널 설치** (수직 직사각형)
3. **뒤채움** (골재 또는 콘크리트)
4. **전면 인장작업** (네일 또는 앵커 긴장)

---

## 4. 단면 구성 (확정)

```
지표면 (비탈면)
  ├─ [1단] 패널 n₁장 + 레벨링 콘크리트 + 보강재 n₁개
  │  소단 폭 b₁
  ├─ [2단] 패널 n₂장 + 레벨링 콘크리트 + 보강재 n₂개
  │  소단 폭 b₂
  ├─ ...
  └─ [최하단] 패널 n_last장 + 레벨링 콘크리트 (두꺼움)
```

**확정 사항:**
- 레벨링 콘크리트 = **각 단마다 독립 기초**
- 보강재 = **패널 1장당 1개**
- 패널 = **수직 직사각형** (경사 없음)
- 기초 형식 = **단별 독립** → 단별 독립 해석

---

## 5. 비탈면 경사선

직선이 아닌 **소단마다 꺾이는 계단형**:
```
1단 배면 상단 → (수평 b₁) → 소단 끝 → (θ경사) → 2단 배면 상단 → ...
```
→ Coulomb 쐐기 배면 경사 β에 해당, 토압 계산에 직접 사용

---

## 6. 해석 단위 (확정)

| 검토 항목 | 단위 | 근거 |
|-----------|------|------|
| 외적안정 (활동·전도·지지력) | **단별 독립** | 단별 레벨링 기초 |
| 보강재→패널 전달 | 단별 독립 | 수평 토압 독립 저항 |
| 전체안정 (원호활동) | **전 구간 통합** | SLOPE/W 외부 입력 |

---

## 7. Phase 02 현장조사 항목

- 외관조사: 패널 균열·박리·변위, 두부 부식, 배수공 막힘, 기초 세굴
- 비파괴: 슈미트해머, 초음파속도, 철근탐사, 탄산화
- 코어 채취: 압축강도 실측
- **Lift-off Test:** 앵커·네일 잔존 인장력 실측 (Phase 03 분기)
- 지반 자료: 시공 당시 보고서
- **★ 단별 레벨링 콘크리트 상태** (균열·세굴·부등침하)

## 8. Phase 03 입력값

| 항목 | 비고 |
|------|------|
| 단면·배근 | 도면 有/無 따라 분기 |
| 재료강도 fck, fy | 열화감소율 반영 |
| PS 잔존력 | Lift-off 실측 or 초기긴장력 × 손실율 |
| 지반정수 γ, φ, c, Ks | 토압 계산 핵심 |
| 앵커·네일 제원 | 길이·직경·간격·경사각·그라우트 강도 |
| **기초 폭 B** | 활동·전도·지지력 계산 핵심 |
| 하중 조건 | 상재하중·지하수위·지진계수 |
