## 핵심 확정 사항 (변경 금지)

1. **H는 역산** — 패널 수 × 패널높이로 계산 (입력값 아님)
2. **레벨링 콘크리트** — 각 단마다 독립 기초 (최하단만이 아님)
3. **보강재** — 패널 1장당 1개 (단당 n개)
4. **패널 형태** — 수직 직사각형 (경사 없음)
5. **해석 단위** — 외적안정은 단별 독립, 전체안정(원호)만 통합
6. **비탈면 경사선** — 소단마다 꺾이는 계단형 (직선 아님)
7. **혼용이 현장 최다** — 상부 네일(PSP) + 하부 앵커(PPP)

## 핵심 타입 (WallSchematic.tsx)

```typescript
export interface TierConfig {
  panels: number      // 단의 패널 수 (1~10)
  bermWidth: number   // 소단 폭 m (최하단 미사용)
}
export interface WallParams {
  height: number        // 역산 H = Σpanels × panelHeight
  length: number; stages: number; slopeAngle: number; wallThick: number
  method: 'PSP' | 'PPP' | 'mixed' | ''
  construction: 'top-down' | 'bottom-up' | 'unknown' | ''
  panelHeight: number   // 패널 1장 높이 (전 단 공통)
  tiers: TierConfig[]   // 길이 = stages
}
```
