## 5. UI 원칙

- 라이트 모드 **고정** (다크모드 없음)
- border-radius: **2~4px** (공학 앱 느낌)
- 그라디언트·광택·과도한 그림자 **금지**
- 모든 계산식에 **KDS 조항번호 명시**
- 입력 가정 전부 **사용자 수정 가능** (블랙박스 금지)

## 7. Phase 01 화면 구성

```
좌측 입력 (40%):
  ① 공법 분류 (공법구분·시공방법)
  ② 자료 보유현황 (보유자료·KDS버전)
  ③ 공통 제원 (연장·단수·경사각·패널두께·패널높이)
  ④ 단별 제원 테이블
     [단번호] [패널수↕] [소단폭↕] [단높이 자동]
     합계 H (역산) / 총 패널 수

우측 모식도 (60%):
  SVG 단면 — 실시간 업데이트
  하단 요약 바: H / L / n / 총패널 / θ / 공법 / 시공
```
