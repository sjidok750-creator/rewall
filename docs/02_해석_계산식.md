# PWAS — 해석·계산식 상세
> Phase 04 개발 시 참조. 모든 수식에 KDS 조항번호 필수.

---

## 1. 외적안정 검토 흐름 (단별 독립, KDS 11 80 20)

```
단 i에 대해:
  ① h_i = 패널 수_i × 패널 높이
  ② Ka 계산 (Coulomb)
  ③ Pa_i = ½ × γ × h_i² × Ka   [KDS 11 80 20 4.3.1]
  ④ 활동 FS = (ΣV × tanδ + c × B) / Pa_i × cosα  ≥ 1.5
  ⑤ 전도 FS = ΣM_저항 / ΣM_전도                    ≥ 1.5
  ⑥ 지지력 FS = q_u / q_max                         ≥ 2.5
  ⑦ 자체파괴 FS ≥ 2.0
```

---

## 2. Coulomb 토압 계수 Ka

```
        sin²(α + φ)
Ka = ─────────────────────────────────────────────
     sin²α × sin(α - δ) × [1 + √(sin(φ+δ)×sin(φ-β) / sin(α-δ)×sin(α+β))]²

  α = 벽체 경사각 (수직이면 90°)
  φ = 내부마찰각
  δ = 벽면 마찰각 (보통 φ/2 ~ 2φ/3)
  β = 배면 경사각 (= 비탈면 θ)

[KDS 11 80 20 4.2.1]
```

---

## 3. 지진 시 토압 — Mononobe-Okabe

```
        (1 - kv) × sin²(α + φ - θ_e)
KAE = ─────────────────────────────────────────────────────────────
      cosθ_e × sin²α × sin(α - δ - θ_e) × [1 + √(sin(φ+δ)×sin(φ-β-θ_e) / sin(α-δ-θ_e)×sin(α+β))]²

  kh = 수평 지진계수
  kv = 수직 지진계수
  θ_e = arctan(kh / (1 - kv))

[KDS 11 80 20 4.2.2]
```

## 4. 각 입력값의 해석 영향

### 단별 높이 h_i
```
Pa_i = ½ × γ × h_i² × Ka
→ h_i² 에 비례 → 가장 중요한 변수
→ h_i = 단 i의 패널 수 × 패널 1장 높이
```

### 소단 폭 b_i
```
① 배면토 자중: W_berm = γ × b_i × h_i  (전도 저항 모멘트 기여)
② 배면 형상: 시행쐐기법 시 계단형 형상 반영
③ 상재하중: b_i < 2 × h_하단 이면 q_i 반영 고려 (FHWA NHI-14-007)
④ 원호활동: 소단 형상이 파괴면에 영향
```

### 비탈면 경사각 θ (= 배면 경사 β)
```
Ka = f(φ, θ, δ, β)  — β = θ
θ 커질수록 Ka 커짐 → Pa 커짐
```

### 기초 폭 B
```
활동: 마찰 저항 = ΣV × tanδ  (B 면적에 비례)
전도: e = B/2 - (ΣM_저항 - ΣM_전도) / ΣV  → e < B/6
지지력: B' = B - 2e  → B 클수록 유리
```

## 5. 보강재→패널 전달 — Phase 04-B (KDS 11 70 15 + FHWA NHI-14-007 §5.3)

> **해석 단위: 패널 1장 (B_panel × h_panel)** — 단위 m 해석 부적합 (PWAS_지침서 §5-4)

### 5-1. 잔존 인장력 확정

```
T_res (실측)  = Lift-off Test 측정값         ← 1차 (Phase 02-D)
T_res (추정)  = T_0 × (1 - 총손실율)         ← 2차 (실측 없을 때)
  총손실율 = Δf_friction + Δf_elastic + Δf_shrink + Δf_creep + Δf_relax
```

### 5-2. 펀칭전단 검토 [KDS 14 20 4.2.10]

```
소요: V_u = T_res

저항:
  b_0 = 2(c_1 + c_2 + 2d)   ← 지압판 주변 임계단면 둘레
  V_c = φ × (1/3)√f_ck × b_0 × d     (φ = 0.75)
  [KDS 14 20 4.2.10 식 (4.2-25)]

요구: V_u ≤ V_c
```

### 5-3. 패널 휨 검토 [KDS 14 30 4.2.1] — 4변 자유·중앙 집중지지 평판

```
하중 모델:
  - 배면 등분포 토압: q = γ × z × Ka  (kN/m²)
  - 중앙 보강재 반력: T_res (집중하중, 내향)

소요 휨모멘트 (FHWA C_F 보정 포함):
  M_u = C_F × q × B_panel² / 8      ← C_F: 1.0~2.0 (하중분배 불확실성)
  [FHWA NHI-14-007 §5.3 Eq. 5-15/16]

형상비 분기:
  B_panel / h_panel ≤ 2 → 평판해석 (양방향 휨)
  B_panel / h_panel > 2 → 보 해석 (단방향, span = B_panel)

저항 (PPC 패널):
  M_n = A_ps × f_ps × (d_p − a/2) + A_s × f_y × (d − a/2)
  [KDS 14 30 4.2.1]

요구: φM_n ≥ M_u   (φ = 0.85)
```

### 5-4. 정착부 지압응력 [KDS 14 30 4.5.2]

```
f_b = T_res / A_bp
허용: f_b ≤ 0.7 × f_ci × √(A/A_bp) ≤ 1.4 × f_ci

A_bp: 지압판 순면적 (mm²)
A   : 지압판 배후 콘크리트 단면적 (mm²)
f_ci: 긴장 시 콘크리트 강도 (MPa)
[KDS 14 30 4.5.2]
```

## 6. PC 패널 단면 검토 — Phase 04-C (KDS 14 30 우선 + KDS 14 20 병용)

> **구조 분류: 부분 프리스트레스트 콘크리트 (PPC)** — RC 단독 검토 불가 (PWAS_지침서 §5-4)

### 6-1. 유효 PS력

```
P_eff = T_res  (Lift-off 실측, 1차)
P_eff = P_i × (1 − 총손실율)  (추정, 2차)
[KDS 14 30 4.4]
```

### 6-2. 균열모멘트 [KDS 14 30 4.3.2]

```
편심 e = 두부 정착점 ~ 패널 단면 도심 거리

M_cr = (f_r + P_eff/A_c + P_eff × e × c / I_c) × I_c / c

f_r = 0.63√f_ck  (콘크리트 파열계수)
```

### 6-3. 극한휨강도 [KDS 14 30 4.2.1]

```
M_n = A_ps × f_ps × (d_p − a/2) + A_s × f_y × (d − a/2)

f_ps = f_pu × (1 − (γ_p/β_1) × (ρ_p × f_pu/f_ck + d/d_p × (ω − ω')))
a   = (A_ps × f_ps + A_s × f_y) / (0.85 × f_ck × B_panel)

요구: φM_n ≥ M_u  (φ = 0.85)
```

### 6-4. 전단강도 [KDS 14 20 6.4]

```
V_n = V_c + V_s

V_c = (1/6)√f_ck × B_panel × d    (전단보강근 없을 때)
[KDS 14 20 6.4.2]

요구: φV_n ≥ V_u  (φ = 0.75)
```

### 6-5. 사용성 검토 [KDS 14 30 4.3.1]

```
노출등급에 따른 허용 인장응력 또는 허용 균열폭 0.20 mm 이하
(부식환경: 0.10 mm 이하)
[KDS 14 30 4.3.1; KDS 14 30 §6.3.2]
```
