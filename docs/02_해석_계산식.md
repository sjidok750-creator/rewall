# PWAS — 해석·계산식 상세
> Phase 04 개발 시 참조. 모든 수식에 KDS 조항번호 필수.

---

## 1. 외적안정 검토 흐름 (단별 독립, KDS 11 80 20)

```
단 i에 대해:
  ① h_i = 패널 수_i × 패널 높이
  ② Ka 계산 (Coulomb)
  ③ Pa_i = ½ × γ × h_i² × Ka   [KDS 11 80 20 4.3.1]
  ④ 활동 FS = (ΣV × tanδ + c × B) / Pa_i × cosα  ≥ 1.5
  ⑤ 전도 FS = ΣM_저항 / ΣM_전도                    ≥ 1.5
  ⑥ 지지력 FS = q_u / q_max                         ≥ 2.5
  ⑦ 자체파괴 FS ≥ 2.0
```

---

## 2. Coulomb 토압 계수 Ka

```
        sin²(α + φ)
Ka = ─────────────────────────────────────────────
     sin²α × sin(α - δ) × [1 + √(sin(φ+δ)×sin(φ-β) / sin(α-δ)×sin(α+β))]²

  α = 벽체 경사각 (수직이면 90°)
  φ = 내부마찰각
  δ = 벽면 마찰각 (보통 φ/2 ~ 2φ/3)
  β = 배면 경사각 (= 비탈면 θ)

[KDS 11 80 20 4.2.1]
```

---

## 3. 지진 시 토압 — Mononobe-Okabe

```
        (1 - kv) × sin²(α + φ - θ_e)
KAE = ─────────────────────────────────────────────────────────────
      cosθ_e × sin²α × sin(α - δ - θ_e) × [1 + √(sin(φ+δ)×sin(φ-β-θ_e) / sin(α-δ-θ_e)×sin(α+β))]²

  kh = 수평 지진계수
  kv = 수직 지진계수
  θ_e = arctan(kh / (1 - kv))

[KDS 11 80 20 4.2.2]
```

## 4. 각 입력값의 해석 영향

### 단별 높이 h_i
```
Pa_i = ½ × γ × h_i² × Ka
→ h_i² 에 비례 → 가장 중요한 변수
→ h_i = 단 i의 패널 수 × 패널 1장 높이
```

### 소단 폭 b_i
```
① 배면토 자중: W_berm = γ × b_i × h_i  (전도 저항 모멘트 기여)
② 배면 형상: 시행쐐기법 시 계단형 형상 반영
③ 상재하중: b_i < 2 × h_하단 이면 q_i 반영 고려 (FHWA NHI-14-007)
④ 원호활동: 소단 형상이 파괴면에 영향
```

### 비탈면 경사각 θ (= 배면 경사 β)
```
Ka = f(φ, θ, δ, β)  — β = θ
θ 커질수록 Ka 커짐 → Pa 커짐
```

### 기초 폭 B
```
활동: 마찰 저항 = ΣV × tanδ  (B 면적에 비례)
전도: e = B/2 - (ΣM_저항 - ΣM_전도) / ΣV  → e < B/6
지지력: B' = B - 2e  → B 클수록 유리
```

## 5. 보강재→패널 전달 (KDS 11 70 15 + FHWA NHI-14-007)

```
잔존 인장력: T_res = T_0 × (1 - 손실율)

펀칭전단 검토:
  V_u ≤ φ × v_c × b_o × d
  b_o = 임계둘레 (두부 지압판 주위)

패널 휨 검토 (4점지지 단순보 가정):
  M_u = T_res × e_s / n_s
  M_u ≤ φ × M_n
```

---

## 6. PC 패널 단면 검토 (KDS 14 20 + KDS 14 30)

```
PS 유효 잔존력: P_eff = P_i × (1 - 총손실율)
  손실 항목: 마찰, 건조수축, 크리프, 릴랙세이션

균열모멘트: M_cr = (f_r + P_eff/A) × S  [KDS 14 30 5.1]
극한모멘트: M_n = A_ps × f_ps × (d_p - a/2)

전단강도: V_n = V_c + V_s  [KDS 14 20 6.4]

정착부 지압응력: f_b = P_i / A_b ≤ φ × 0.85 × f'_c
```
